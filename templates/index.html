Renumerador de Recibos

Envie um arquivo Word ou um arquivo ZIP com vários recibos. O sistema identifica o último número de cada arquivo, continua a sequência automaticamente e devolve os documentos prontos para uso.

Numeração automática Localiza “NOTA DE BALCÃO”, preserva zeros à esquerda e continua a sequência no próprio arquivo.
Mesmo layout O sistema altera somente a numeração dos recibos, mantendo a estrutura original do documento.
Retorno imediato Para um arquivo, o retorno já vem em Word. Para vários, o resultado é entregue em ZIP com relatório.
Dica: para processar várias empresas de uma vez, compacte os arquivos em ZIP antes de enviar.
O arquivo de saída mantém o mesmo nome do original. Em uploads múltiplos, o resultado vem acompanhado de um relatório simples de processamento.
